---
name: No pedir permiso para ejecutar comandos
description: El usuario no quiere que se le pida permiso antes de ejecutar comandos de terminal
type: feedback
---

No pedir confirmación antes de ejecutar comandos de bash, npm, git u otras herramientas de terminal. Ejecutar directamente.

**Why:** El usuario lo indicó explícitamente para agilizar el flujo de trabajo.

**How to apply:** Siempre ejecutar comandos sin preguntar primero, a menos que sea una acción destructiva e irreversible (borrar archivos, force push, etc.).
